

Должен уметь:
- Определять момент проезда по трубе
- По двум проездам в течение XX секунд, либо отсутствию второго события в
  течение этого времени после первого — детектирование события "ТС проехало"
- хранение времени в RTC
- запись в EEPROM:
	- событий в формате "время, скорость ТС"
	- "сырых" событий (только время) — при необходимости
- связь через UART:
	- сообщения о событиях
	- приём команд
- команды:
	- вывести содержимое памяти
	- стереть память
	- вывести конфигурацию
	- установить значение переменной
	- вывести статистику по событиям из памяти
